Getting Started with the Marvel Comics API

This page will show you how to sign up with Marvel and get a Key.

Creating an Account with Marvel

The first step to working with the Marvel Comics API is to create an account.
Follow this link and sign up:


Getting your API Keys

Now that you have an account head over to the developer portal and get your API keys.
You can find your keys in your developer account:

Once you are in your developer account, you will find your public and private API keys. As stated before, we are creating a client-side application
so for now, we will only be concerned with the public key.

You will also need to list any domains that can make calls to the Marvel Comics API using your keys.
Because we are making a front-end application, we will only need to authorize our local IP/localhost.

This is the basics of getting started with the Marvel Comics API. If you would like more information, head over to Marvel for a detailed documentation.
